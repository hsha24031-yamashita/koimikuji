恋みくじアプリ (最強版)
ボタンをクリックするだけで、今日の恋愛運を占うことができるシンプルなウェブアプリケーションです。
特徴
直感的な操作: 「おみくじを引く」ボタンを押すだけのシンプルなUI。

レスポンシブなデザイン: ピンクを基調とした、親しみやすいデザイン。

軽量: 外部ライブラリを使用せず、標準的なウェブ技術のみで構成されています。
ファイル構成
プロジェクトは以下の3つのファイルで構成されています。

index.html: アプリの構造（タイトル、ボタン、結果表示エリア）。

style.css: アプリの見た目（背景色、ボタンのスタイル、フォント設定）。

script.js: おみくじのロジック（ランダムな運勢の選出と表示）。
使い方
1.すべてのファイル（index.html, style.css, script.js）を同じフォルダに保存します。

2.ブラウザで index.html を開きます。

3.「おみくじを引く」ボタンをクリックすると、画面に運勢が表示されます。
技術スタック
HTML5: 構造の定義

CSS3: スタイリング、ホバーエフェクト

JavaScript (ES6): イベントリスナー、配列操作、乱数生成

カスタマイズ方法
script.js 内の fortunes 配列の内容を書き換えることで、運勢の種類やメッセージを自由に変更することができます。
